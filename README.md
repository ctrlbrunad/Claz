**🏋️‍♀️ Use Claz | Landing Page de Moda Fitness**

Landing page desenvolvida para uma loja virtual de moda fitness, com foco em apresentação visual, identidade de marca e conversão de usuários.

🔗 Acesse o projeto online: https://useclaz.netlify.app/

📌 Sobre o projeto

Este projeto consiste em uma landing page moderna e responsiva criada para representar uma marca de moda fitness.
O objetivo foi construir uma experiência visual atrativa, com foco em destacar produtos, incentivar a navegação e gerar interesse do usuário.

A página foi desenvolvida utilizando tecnologias web fundamentais, priorizando organização, performance e boa experiência em diferentes dispositivos.

🚀 Tecnologias utilizadas
HTML5
CSS3
JavaScript
🎯 Objetivos do projeto
Criar uma landing page com identidade visual forte
Trabalhar conceitos de responsividade
Praticar estruturação de código front-end
Desenvolver um layout voltado para conversão
📱 Responsividade

A landing page foi construída com foco em adaptação para diferentes tamanhos de tela, garantindo uma boa experiência tanto em desktop quanto em dispositivos móveis.

Possíveis melhorias futuras:
Integração com backend (ex: carrinho de compras)
